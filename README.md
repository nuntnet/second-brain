# Second Brain

Personal knowledge base managed by Obsidian + Hermes AI.

## Structure (PARA)

| Folder | ใช้สำหรับ |
|--------|----------|
| `Daily/` | Daily notes, journal, วันนี้ทำอะไร |
| `Projects/` | งานที่กำลังทำอยู่ มี deadline |
| `Areas/` | ความรับผิดชอบระยะยาว (career, health, finance) |
| `Resources/` | ความรู้, research, reference |
| `Archive/` | เสร็จแล้ว / ไม่ active แล้ว |

## AI Integration

Hermes สามารถ:
- อ่าน/เขียน notes ผ่าน MCP
- บันทึกสรุป conversation
- ค้นหา notes ก่อนตอบคำถาม
- สร้าง daily note อัตโนมัติ
